# Excel Orders Analysis Project

## Project Overview
This Excel project provides a complete overview of orders, customer segments, and product performance.  
It includes interactive trackers and built-in hyperlinks that allow quick navigation between all sheets for easy and efficient analysis.

## Sheets Included
- **Orders Overview**: High-level summary of orders and key metrics
- **Orders Tracker**: Detailed tracking of all orders
- **Customer Segmentation**: Analysis of customers based on behavior and performance
- **Product Overview**: Product performance and contribution analysis

## Key Features
- Clean and structured data
- Pivot tables for analysis
- Charts for visualization
- Hyperlinks for easy navigation between sheets

## Tools Used
- Microsoft Excel

## Analysis Steps
- Data cleaning and validation
- Order tracking and aggregation
- Customer segmentation analysis
- Product performance evaluation
- Dashboard-style overview for insights

## Key Insights
- Clear visibility of order trends
- Identification of high-value customers
- Understanding top-performing products
- Improved navigation and usability through hyperlinks

## Purpose
This project demonstrates practical Excel skills for data analysis, reporting, and business decision support.
